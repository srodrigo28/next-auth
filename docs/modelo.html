Módulo Inicial

Introdução ao Python com Django

Django é um framework web em Python conhecido por entregar uma estrutura mais completa e opinativa. Ele é muito usado para sistemas robustos, painéis administrativos, backends corporativos e aplicações com crescimento organizado.

Por que Django?

Ele já traz muito do que um projeto real precisa: ORM, admin, autenticação e organização por apps.

Perfil do framework

Django é menos minimalista que Flask e mais completo logo de saída, o que acelera times que querem padrão.

Onde ele aparece

ERPs, intranets, sistemas internos, plataformas de conteúdo, autenticação e backends mais estruturados.

PASSO 2

O que é Django na prática

Django é um framework full-stack para back-end web. Ele organiza o projeto em componentes como URLs, views, templates, models e admin, tudo pensado para produtividade com padrão.

Exemplo mínimo

Criando o projeto

pip install django
django-admin startproject config .
python manage.py runserver

Projeto e apps

Em Django, o projeto principal cuida da configuração geral, enquanto os apps dividem responsabilidades do sistema.

Projeto

Apps

Estrutura

startapp

python manage.py startapp usuarios
PASSO 4

URLs e views

No Django, URLs direcionam a requisição para views, que processam a lógica e devolvem a resposta.

Roteamento

path + view

from django.urls import path
from . import views

urlpatterns = [
    path("", views.home, name="home"),
]
PASSO 5

Templates e páginas web

Templates permitem renderizar HTML com dados vindos da view, usando a template language do Django.

Renderização

render

from django.shortcuts import render

def home(request):
    return render(request, "home.html", {"nome": "Ana"})
PASSO 6

Models e ORM

O ORM do Django permite representar tabelas como classes Python e consultar dados sem escrever SQL o tempo todo.

Banco de dados

Model básico

from django.db import models

class Produto(models.Model):
    nome = models.CharField(max_length=120)
    preco = models.DecimalField(max_digits=10, decimal_places=2)
PASSO 7

Admin Django

O painel administrativo é uma das funcionalidades mais marcantes do Django. Ele acelera muito CRUDs internos e operação de dados.

Admin

Registrando model

from django.contrib import admin
from .models import Produto

admin.site.register(Produto)
